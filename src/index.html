galpy is a Python package for galactic dynamics.

Easy to get started with, get advanced galactic dynamics at your fingertips

Get started with integrating orbits in a Milky-Way potential in a few lines of code in minutes. Or build your own galactic models using a flexible set of components. From simple orbit integration to fast action-angle calculations, galpy has you covered.

Simple installation on all major operating systems

Whether you are on a Linux, Mac, or Windows system, installation is quick and painless. Use miniconda for a single-line installation of galpy and all of its dependencies or use pip for installing the latest release or the bleeding-edge master branch with all of the latest in-development features.

Extensive documentation and many ways to get help

All of galpy's features are documented with a rich set of examples in the quick-start guide and a full library reference guide. If you run into problems, open an issue on galpy's GitHub page, join the galpy slack community (join here), or email the code's maintainer. We're always happy to help!

Open-source, continuously-integrated, with >99.8% test coverage

Fully open-source development since day 1. All of galpy's development happens on GitHub, from simple maintenance to bleeding-edge new features. All code updates and pull requests are built and tested using continuous integration provided by travis and appveyor. With test coverage exceeding 99.8%, galpy is as reliable as code gets.
